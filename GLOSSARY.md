## 📖 Alphabetical Glossary

| CONCEPT | MALL ANALOGY | DEFINITION | OFFICIAL DOCS |
| :--- | :--- | :--- | :--- |
| **API Server** | The Management Office | The single source of truth. Every request to change the mall must go through this office. | [Link](https://kubernetes.io/docs/concepts/overview/components/#kube-apiserver) |
| **ClusterRole** | The Mall-Wide Authority | Rules that apply to the **entire Mall (Cluster)**, like access to building blueprints (Nodes). | [Link](https://kubernetes.io/docs/reference/access-authn-authz/rbac/#role-and-clusterrole) |
| **ConfigMap** | The Shop Manual | A set of non-sensitive configuration rules for a shop (e.g., color scheme, open hours). | [Link](https://kubernetes.io/docs/concepts/configuration/configmap/) |
| **CronJob** | Scheduled Maintenance | Repeating tasks (e.g., cleaning the floors every night at 2 AM). | [Link](https://kubernetes.io/docs/concepts/workloads/controllers/cron-jobs/) |
| **DaemonSet** | Floor Security | Ensures exactly one worker (Pod) remains on every single building (Node) at all times. | [Link](https://kubernetes.io/docs/concepts/workloads/controllers/daemonset/) |
| **Deployment** | The Store Manager | Ensures that the correct number of workers (Pods) are always on duty. | [Link](https://kubernetes.io/docs/concepts/workloads/controllers/deployment/) |
| **emptyDir** | The Day-Use Locker | Temporary storage shared between workers in the same shop (Pod). Wiped when the shop closes. | [Link](https://kubernetes.io/docs/concepts/storage/volumes/#emptydir) |
| **EndpointSlice** | The Digital Staff List | The real-time list of which workers are currently at their desks and ready to help. | [Link](https://kubernetes.io/docs/concepts/services-networking/endpoint-slices/) |
| **Gateway API** | The Smart Transit Hub | **(Modern)** A modular way to manage traffic using roles (Architect, Admin, Developer). | [Link](https://kubernetes.io/docs/concepts/services-networking/gateway/) |
| **Ingress** | The Main Gate Guard | **(Legacy)** A rulebook that tells the entrance guard where to send customers based on their request. | [Link](https://kubernetes.io/docs/concepts/services-networking/ingress/) |
| **Job** | Temporary Contractor | A task that runs until it is finished (e.g., a one-time inventory count). | [Link](https://kubernetes.io/docs/concepts/workloads/controllers/job/) |
| **kubeconfig** | The Owner's ID Badge | The digital certificate that proves who you are and grants access to the Management Office. | [Link](https://kubernetes.io/docs/concepts/configuration/organize-cluster-access-kubeconfig/) |
| **kubectl** | The Management Terminal | The tool used to send commands to the Management Office. | [Link](https://kubernetes.io/docs/reference/kubectl/) |
| **Namespace** | The Individual Shop Space | A logical boundary that keeps one shop's resources separate from another. | [Link](https://kubernetes.io/docs/concepts/overview/working-with-objects/namespaces/) |
| **NetworkPolicy** | Internal Shop Security | Rules that define which shops (Pods) can talk to each other. | [Link](https://kubernetes.io/docs/concepts/services-networking/network-policies/) |
| **Node** | The Mall Building | A physical or virtual machine where the shops (Pods) are actually located. | [Link](https://kubernetes.io/docs/concepts/architecture/nodes/) |
| **PersistentVolume** | The Warehouse Vault | A permanent storage unit in the basement that stays even if the shop is replaced. | [Link](https://kubernetes.io/docs/concepts/storage/persistent-volumes/) |
| **Pod** | The Individual Employee | The smallest unit in Kubernetes; a worker that runs your application. | [Link](https://kubernetes.io/docs/concepts/workloads/pods/) |
| **RBAC** | The HR Policy | Rule-Based Access Control: deciding who can do what (Read, Write, Delete) in the mall. | [Link](https://kubernetes.io/docs/reference/access-authn-authz/rbac/) |
| **Role** | The Local Shop Permit | A list of "Can-Do" rules restricted to a single shop (Namespace). | [Link](https://kubernetes.io/docs/reference/access-authn-authz/rbac/#role-and-clusterrole) |
| **Secret** | The Manager's Safe | Storage for sensitive information like passwords or API keys. | [Link](https://kubernetes.io/docs/concepts/configuration/secret/) |
| **Service** | The Storefront Sign | A stable entry point or "phone extension" that stays the same even if workers move. | [Link](https://kubernetes.io/docs/concepts/services-networking/service/) |
| **ServiceAccount** | An Employee ID Badge | The identity used by a worker (Pod) to talk to the Management Office. | [Link](https://kubernetes.io/docs/concepts/security/service-accounts/) |
| **StatefulSet** | Specialized Boutique | For apps that need stable "identities" and persistent storage (like a database). | [Link](https://kubernetes.io/docs/concepts/workloads/controllers/statefulset/) |
